# Nishana Couture — Static Website

A luxury boutique fashion website for **Nishana Couture**, Chennai.

## Structure

```
nishana-couture/
├── index.html          ← Main page
├── assets/
│   ├── css/style.css   ← All styles
│   └── js/script.js    ← Interactivity
└── README.md
```

## Deploy to GitHub Pages

1. Create a new GitHub repo (e.g. `nishana-couture-site`)
2. Upload all files preserving the folder structure
3. Go to **Settings → Pages**
4. Set source: **Deploy from branch → main → / (root)**
5. Your site will be live at: `https://yourusername.github.io/nishana-couture-site/`
